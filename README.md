# tri5g-webrtc-camera-client

```
python -m venv venv
```

```
pip install --no-cache-dir -r requirements.txt
```

```
python main.py --camera "Integrated Camera" --fps 30 --resolution 640x480
```

```
Command line values:
--url [URL for WebRTC connection, eg: "http://localhost:9000/offer"] 
--camera [Camera device name, eg: Integrated Camera] 
--resolution [resolution, default: 800x600]
--fps [framerate, default: 30] 
--force-h264 [force h264, default: False]
--username [username for server auth]
--password [password for server auth]
```
