```
Command line values:
--camera [Camera device name, eg: Integrated Camera] 
--key [Key for WebRTC server, optional]
--url [URL for WebRTC connection, eg: "http://localhost:9000/offer"] 
--resolution [resolution, default: 800x600]
--fps [framerate, default: 30] 
--force-h264 [force h264, default: False]
```
